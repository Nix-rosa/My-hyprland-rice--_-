- 📚 Browse manga con portadas (timg)
- 🎬 Browse anime con video preview
- 🎨 ASCII logos únicos por serie
- 🔍 Búsqueda global
- 🎲 Modo aleatorio
- 📊 Estadísticas detalladas
-📖 Funciona con  fomato de manga cbz y animes de formato mp4,mkv
